# **AUTO**COMPARA — Site para complementação do trabalho

Projeto Final de Curso · Turma 2 · Análise de Dados · Generation Brasil

## Finalidade
Plataforma Web interativa de suporte à tomada de decisão para aquisição de automóveis com base em análise multicritério.

A plataforma não é apenas um site estático. Ela une Engenharia de Dados (ao consumir e tratar dados reais de uma planilha do Google em nuvem), Ciência de Decisão (com a matriz matemática de pesos) e Design Centrado no Usuário (UI/UX) (com filtros dinâmicos e navegação responsiva).

---

## ⚠️ Nota sobre os dados

Os dados de preço, consumo, potência e demais especificações são **estimativas** baseadas em fontes públicas (FIPE, Webmotors, fabricantes) e podem conter **inconsistências**. O site já exibe um aviso ao usuário sobre isso. Recomenda-se atualizar os dados com fontes primárias antes da apresentação final.
